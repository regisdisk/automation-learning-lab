# What Worked

## Event → State pattern

Moving from:

Notification
→ Human

to:

Event
→ State
→ Action

reduced coupling and improved extensibility.

---

## Home Assistant as action provider

Replacing Virtual Button introduced:

- lower operational dependency
- observable state
- reusable automation

Unexpected gain:

Home Assistant became an action provider with observable state.

---

## Documentation-first closure

Stopping implementation to document:

- workflow
- architecture
- recovery
- backups

reduced future operational risk.

---

## Treating ignored files as protected

Incorrect assumption:

## Browser authentication diagnosis

Password reset failed.

Session persistence created false diagnosis.
