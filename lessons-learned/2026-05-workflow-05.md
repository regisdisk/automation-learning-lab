# What Worked

## Event → State pattern

Moving from:

Notification
→ Human

to:

Event
→ State
→ Action

reduced coupling and improved extensibility.

---

## Home Assistant as internal state engine

Replacing Virtual Button introduced:

- lower operational dependency
- observable state
- reusable automation

Unexpected gain:

Home Assistant became an event consumer.

---

## Documentation-first closure

Stopping implementation to document:

- workflow
- architecture
- recovery
- backups

reduced future operational risk.

---
