Date:
2026-05

Release:
v0.5.0

Workflow:
Email Driven Home Automation

---

## Objective

Convert email events into automated actions through a reusable event pipeline.

---

## Architecture

Gmail
→ Google Apps Script
→ n8n
→ Home Assistant

---

## What Worked

- Webhook-based event ingestion
- Conditional routing in n8n
- Home Assistant helper state
- State cleanup automation
- End-to-end validation

---

## What Failed

- Gmail IMAP approach
- Alexa Media Player authentication
- Assuming HTTP 200 meant completed action

---

## Decisions Taken

Keep:

Event
→ Decision
→ State
→ Action

Avoid:

Email
→ Direct Device Action

---

## Mental Models Reinforced

Producer ≠ Publisher

Decision ≠ Execution

HTTP Accepted ≠ Outcome Confirmed

State ≠ Automation

---

## Technical Debt

- Alexa integration incomplete
- No delivery confirmation from external notifications
- Authentication flow still manual
