# Automation Learning Lab

Hands-on experiments exploring workflow automation, APIs, containers and cloud deployment.

## Purpose

This repository documents practical experiments designed to learn and apply:

- Workflow automation
- API integration
- Containerized environments
- Cloud deployment
- Data transformation
- Event-driven execution

This is not intended to be a production-grade platform.

The goal is to progressively evolve from small learning exercises into reusable automation patterns.

---

## Environment

Infrastructure:
- Oracle Cloud Infrastructure (OCI)
- Ubuntu 22.04
- Docker
- Docker Compose

Automation:
- n8n
